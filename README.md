# SnakeGame
Basic snake game I coded in python as a weekend project
# Usage
Clone repo into a folder  
```
git clone [Link]
```
To download required libraries:   
```
pip3 install pygame
```
To run the game:    
```
python3 game.py
```
# Controls
WASD to move snake  
Red blocks is regular food  
Blue blocks is a speed boost and 5 food
